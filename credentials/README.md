# Credentials Format

## Algorithmia

File: `algorithmia.json`

```
{
    "apiKey": String
}
```

## IBM Watson - Natural Language Understading

File: `watson-nlu.json`

```
{
  "apikey": String,
  "iam_apikey_description": String,
  "iam_apikey_name": String,
  "iam_role_crn": String,
  "iam_serviceid_crn": String,
  "url": String
}
```
